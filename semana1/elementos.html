HTML

Es un lenguaje marcado de hipertexto que define la estructura y el significado de un contenido web, HTML es el más básico componente de esta. Para crear una página web, además de HTML se utilizan otras tecnologías para describir la apariencia y presentación de la página. HTML hace uso de “marcado”, el cual incluye “elementos” para poder etiquetar texto, imágenes y otros contenidos para mostrarlo en un navegador web.

< !DOCTYPE html >

Representa el tipo de documento, anteriormente verificaba automáticamente errores y otras cosas de utilidad. En la actualidad se considera como un artefacto antiguo sin importancia, pero debe ser incluido para que funcione adecuadamente.

< html > < /html >

Es un elemento conocido como el elemento raíz (root element), este elemento encierra todo el contenido de una página.

< head > < /head>

Es un elemento que actúa como contenedor de todo lo que incluye una pagina web HTML, todo aquello que no es contenido visible por los visitantes de la página. Incluye cosas como:

  • Palabras clave (keywords).
  • Una descripción de la página que quieres que aparezca en resultados de búsquedas.
  • Código CSS para dar estilo al contenido.
  • Declaraciones del juego de caracteres, etc.

< title > < /title>

Es un elemento que establece el título de una página, hablamos de un título que aparecerá en la pestaña o en la barra del titulo del navegador cuando la página se cargue, se usa para describir la pagina cuando esta es añadida a los marcadores o como favorita.

< body > < /body >

Es un elemento que representa el contenido de un documento HTML, únicamente se puede realizar un elemento en un documento y debe ser el segundo elemento de un elemento . Encierra todo el contenido que se muestra a los usuarios web que visitan la página ya sea texto, imágenes, videos, juegos, pistas de audio reproducibles, y demás.

< h1 > < /h1 >

Es un elemento de encabezado que permite especificar algunas partes del contenido son encabezados, o subencabezados del contenido. Un documento HTML puede tener al igual que los libros un titulo principal y que a su vez puede tener títulos por cada capítulo individual, y subtítulos dentro de ellos.

< p > < /p >

Es un elemento que se utiliza para encerrar párrafos de texto, si se quiere especificar de que se trata un párrafo, podemos encerrar el texto en esta etiqueta de párrafo.

  • La etiqueta de apertura: consiste en el nombre del elemento (en este caso, p), encerrado por paréntesis angulares (< >) de apertura y cierre. Establece dónde comienza o empieza a tener efecto el elemento en este caso, dónde es el comienzo del párrafo.
  • La etiqueta de cierre: es igual que la etiqueta de apertura, excepto que incluye una barra de cierre (/) antes del nombre de la etiqueta. Establece dónde termina el elemento en este caso dónde termina el párrafo.
  • El contenido: este es el contenido del elemento, que en este caso es sólo texto.
  • El elemento: la etiqueta de apertura, más la etiqueta de cierre, más el contenido equivale al elemento.

< h2 > < /h2 >

Lo usamos cuando queremos poner subtítulos en nuestra página.

< h3 > < /h3 >

Lo usamos para dividir el contenido que se está tratando en esa subsección, además de buscar palabras clave derivadas.

< body link alink vlink >

Nosotros podemos definir el color del link con la etiqueta link, el color del link cuando está activo con la etiqueta alink y el color del link cuando ya se ha visitado con la etiqueta vlink.

< body leftmargin topmargin rightmargin bottommargin >

  • Leftmargin: Es para definir el margen del lado izquierdo.
  • Topmargin: Es para el margen de la parte de arriba.
  • Rightmargin: Es para el margen del lado derecho.
  • Bottommargin: Es para el margen de la parte de abajo.

< body text >

Con esto nosotros podemos definir el color la tipografía de nuestra página web.

< body background >

Con esto nosotros podemos definir una imagen de internet para fondo de la página.

< body bgcolor >

Con esto nosotros podemos definir el color de fondo de la página.

< meta >

Sirve para incluir información acerca de las características de la página, y de esta forma se tengan palabras clave que ayuden al navegador a encontrar tu página.

< script >

Este sirve para incluir código en lenguajes de script, los cuales el navegador es muy fácil de interpretar.

< style >

Este es ocupado para incluir estilos CSS que permiten incluir de colores, bordes, imágenes de fondo, entre otros, a los elementos de la página.

< base >

Este sirve para indicar la URL principal en caso de que la URL se relacione con otra dentro de la página web.

< html lang='es' >

Es la raíz de un documento HTML o XHTML. Tiene como parametro lang, que se utiliza para identificar el idioma del contenido.

< link >

Con esto nosotros podemos indicar con cual se relaciona el documento html, ya sea otro archivo o recurso externo.
